## 📖 О проекте

Каждый час в Белое море попадает **3 000 частиц пластика**. Это исследование визуализирует путь мусора от архангельских улиц до арктических берегов — через реку, приливы и течения Белого моря.

Проект включает интерактивную карту, статистику, фотогалерею найденных предметов и мини-игру по сортировке мусора.

---

## ✨ Возможности

### 🗺️ Интерактивная карта
- Маркеры источников загрязнения и точек уборок
- Визуализация пути мусора с течениями
- Анимация движения частиц пластика

### 📊 Данные и факты
- Статистика по Белому морю
- Глобальные факты о пластиковом загрязнении
- Инфографика морфологического состава мусора

### 🎮 Игра «Очисти побережье»
- Бесконечная сортировка мусора по контейнерам
- Обучение через геймификацию
- Счёт очков и ошибок

### 📷 Галерея находок
- Фотографии предметов с берегов Белого моря
- Фильтрация по материалу
- Подробная информация о каждой находке

## 🚀 Запуск

```bash
# Клонирование репозитория
git clone https://github.com/your-username/trash-current.git
cd trash-current

# Запуск через Python
python3 -m http.server 8080

# Или через Node.js
npx serve .
```

Откройте в браузере: [http://localhost:8080](http://localhost:8080)
